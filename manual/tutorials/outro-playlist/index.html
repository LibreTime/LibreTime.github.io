How to setup an Outro Playlist

This how to describes the steps involved in setting up an outro playlist that will schedule promos without interrupting any tracks followed by a ambient musical bridge for all autoloading playlists to avoid dead air.

Audience: Program Manager (and Admin to set Outro Playlist)

Adding the Promo Fill smart block

  1. Click Smart Blocks
  2. Click the blue + New button
  3. Type in a name for the Smart Block (for instance Promo Fill)
  4. Select Genre and the modifer of is and then type in promo in the 3^rd^ box.
  5. Under Limit to: we select time remaining in show
  6. To match on additional types of tracks based upon Genre, click + New Modifier under Genre and select is and type the “name of genre” to match on.
  7. Click preview to review and confirm tracks match your criteria below.
  8. Click Save

Adding interuptible Ambient Fill smart block

  1. Click Smart Blocks
  2. Click the blue +New button
  3. Type in a name for the Smart Block (for instance Ambient Fill)
  4. Now we select Genre and the modifer of is and then type in ambient fill in the 3^rd^ box.
  5. Under Limit to: we select time remaining in show
  6. Click Advanced options drop down
  7. Check box next to Allow last track to exceed time limit
  8. Click preview to review and confirm tracks match your criteria below.
  9. Click Save

Create the Outro Playlist to hold the smart blocks

  1. Click Playlists
  2. Click + New
  3. Type in the name “Outro Playlist”
  4. Click on Smart Blocks
  5. Click box next to Promo Fill smart block and click + Add to current playlist or drag it to the bottom of the Outro playlist
  6. Click box next to Ambient Fill smart block and click + Add to current playlist or drag it to the bottom of the Outro playlist
  7. Make sure the Promo Fill is above the Ambient Fill smart block in the playlist
  8. Click Save

Setting the station-wide Outro playlist

  1. Click Settings
  2. Click General
  3. Scroll down to Outro Playlist and with the drop down select the newly created Outro Playlist
  4. Scroll down or up on the form and click Save.

Now every autoloading playlist show will have tracks scheduled to fill any potential dead air.
